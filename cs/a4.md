```

    1. a slight change in data can change the whole hash
    2. can't detect new or unknown malwares
    3. it is uneffective against polymorphic or packed malware

How attackers bypass it
    1. they can slightly modify the file to change its hash
```

```
Obfuscation

Packing
    it is the process of compression or encryption of executable file which is used to wrap code to help it evade signature based detection
```
